# PhysioConnect
PhysioConnect is a healthcare platform that connects patients with verified physiotherapists.

The platform helps users easily search, connect, and book appointments with trusted physiotherapists while providing physiotherapists with a digital platform to manage their services.



## Features

### User Features
- User authentication
- Search physiotherapists
- View physiotherapist profiles
- Book appointments
- WhatsApp & Call integration
- Ratings & reviews

### Physiotherapist Features
- Profile creation
- Degree/ID upload
- Availability status
- Appointment management
- Verification status

### Admin Features
- Verification queue
- Approve/reject physiotherapists
- Manage reports & complaints
- Monitor bookings

---

## AI-Assisted Verification

Physiotherapists upload degree certificates or student IDs.

The system uses:
- OCR text extraction
- AI-assisted validation
- Admin review queue

Profiles are created instantly while verification runs in the background.

---

## Ranking System

Verified and highly rated physiotherapists appear higher in search results based on:
- Ratings
- Completed sessions
- Response speed
- Verification bonus
